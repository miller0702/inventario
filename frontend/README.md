
## Frontend
- **Framework**: Vue 3
- **Librería de componentes**: Element Plus
- **Estado global**: Vuex
- **Dependencias**: `package.json`

## Instrucciones para el Frontend

1. Instalar las dependencias:

   ```npm install```

2. Ejecutar la aplicación:

   ```npm run dev```
